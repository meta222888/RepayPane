# RelayPane 功能需求方案书 (PRD)

## 一、 项目概述
RelayPane 是一款使用 Go 语言开发的轻量级服务器文件管理工具，旨在实现类似 WinSCP 的核心双栏文件传输与管理功能。软件专注于高效的 SFTP 连接、直观的拖放操作以及便捷的远程文件在线编辑，提供极简、流畅的用户体验。

---

## 二、 技术栈选型
* **前端 UI 框架**：Wails (基于 Web 技术栈，推荐 Vue3 + Nuxt3 或 React，便于实现复杂的拖放、快捷键监听及现代 UI 体验)
* **后端核心**：Go 语言 (`golang.org/x/crypto/ssh`, `github.com/pkg/sftp`)
* **本地文件系统**：Go 原生 `os`, `path/filepath` 包

---

## 三、 界面布局与视觉架构 (UI/UX)
整体采用经典的 **WinSCP 双栏对开** 布局，分为以下五个核心区域：

### 1. 顶部菜单栏 (Menu Bar)
* **文件 (File)**：新建会话、断开连接、退出。
* **功能 (Features)**：刷新、设置。
* **关于 (About)**：软件版本、版权信息。

### 2. 上方工具栏 (Toolbar)
* **左侧上方（本地）**：
  * 本地路径下拉选择框（支持切换 C 盘、D 盘等盘符，或选择常用历史目录）。
  * 按钮：返回上一级、新建文件夹。
* **右侧上方（远程）**：
  * 远程路径面包屑导航（支持点击路径某一段直接跳转）。
  * 按钮：返回上一级、新建文件夹。

### 3. 主体双栏区域 (Main Panels)
* **左栏（本地文件系统）**：表格/列表视图。展示：文件名、大小、修改时间。
* **右栏（远程文件系统）**：表格/列表视图。展示：文件名、大小、权限（Permissions）、修改时间、属组。

### 4. 底部状态栏 (Status Bar)
* 展示当前连接状态、传输进度条（百分比、速度）、队列剩余任务数。

## 四、 核心功能需求

### 1. 服务器会话管理 (Session Management)
* **会话存储**：支持保存多个服务器配置。配置信息（主机名、端口、用户名、密码/私钥路径、别名）以加密形式存储在本地 JSON 文件中。
* **快速连接**：启动软件时，默认弹出会话管理器窗口，双击即可发起 SSH/SFTP 连接。

### 2. 文件传输与双向拖放 (File Transfer & Drag-and-Drop)
* **双向拖拽**：
  * 从左栏（本地）拖拽到右栏（远程） $\rightarrow$ **触发上传**。
  * 从右栏（远程）拖拽到左栏（本地） $\rightarrow$ **触发下载**。
* **外部拖入**：支持从 Windows 操作系统资源管理器直接拖拽文件/文件夹到软件的本地或远程列表。
* **覆盖提示**：遇到同名文件时，弹出提示框：覆盖、跳过、重命名。

### 3. 远程文件在线编辑 (Remote File Editing)
* **双击打开**：双击右栏（远程）的文本文件：
  1. Go 后端首先检查文件大小。
  2. **大文件拦截**：若文件 **> 10MB**，弹窗提示：*“当前文件较大，直接打开可能会导致卡顿，是否确认下载并打开？”*
  3. **临时缓存**：若大小允许，自动下载到本地系统临时目录（Temp），并在内置编辑器（或调用系统默认编辑器）中打开。
* **实时保存 (Ctrl + S)**：
  * 监听编辑器内的 `Ctrl + S` 快捷键。
  * 触发时，前端通知 Go 后端，后端立即发起**静默 SFTP 上传**，将修改后的临时文件覆盖回远程服务器。
  * 状态栏提示：*“文件已成功同步至远程服务器”*。

# Changelog

## 1.0.2 — 2026-06-06

### 新功能

- **远程极速删目录**：删除远程文件夹时在服务器执行 `rm -rf`，超大目录几乎瞬间完成；WinSCP 等工具仍通过 SFTP 逐文件删除，面对 GB 级目录往往要几分钟甚至更久。

### 改进

- 远程/本地删除改为后台异步，删除大目录时界面不再卡死或未响应。
- 传输状态栏区分「上传到服务器」与「从服务器下载」；修复进度条不随传输更新的问题。
- 文件列表：空白处左键/右键可用；重命名后点击空白或其它区域可正常退出。
- 拖放文件时使用自定义复制光标；顶部栏去掉与系统标题重复的应用名，主菜单左对齐。
- 选中后稍等再双击可正常进入目录，不再误触发行内重命名。

### 修复

- 文件列表空白区需 resize 窗口后才响应点击的问题。
- 双击选中项与慢速二次点击改名的冲突。

## 1.0.1 — 2026-06-06

### 新功能

- **文件多选**：本地/远程文件列表支持 **Ctrl+A** 全选、**Ctrl+点击** 多选；可复制、删除、拖放到对面面板批量传输。
- **重命名**：慢速双击文件名进入行内重命名；右键菜单新增 **重命名**（单选时可用）。
- **云端同步**：设置 → **云端同步**，AES 加密备份/恢复服务器列表（API 密钥与加密口令本地保存）。
- **检查更新**：关于 → **检查更新**，从 GitHub Releases 比对版本并打开下载页。
- **文本编码**：打开/保存编辑器时自动检测 UTF-8、GBK 等编码。
- **图片与二进制**：PNG 等图片双击用系统默认程序打开；非文本二进制不再误入编辑器。
- **网络信息**：实时带宽速率、路由与端口输出可框选复制；自动刷新改为原地更新，无闪烁。
- **磁盘空间**：打开窗口即显示 **读取中……**，加载完成后展示占用卡片。

### 改进

- 修复英文界面字母下伸（g、y 等）被裁切的问题。
- 修复网络流量列表被压成细条、端口区多余空行。
- 文件列表空白处可右键打开菜单；菜单左右留白加宽。
- 确认对话框改为画布 overlay，子窗口不再被主窗口挡住；菜单内边距加大。
- 服务器标签栏与「+连接」行高度压缩；绿色强调按钮 hover 不再变灰。
- 拖放复制仅限本地 ↔ 远程跨栏，同侧不再误触发。
- 编辑器：Ctrl+S 保存提示、还原自文件、状态栏保存结果与时间戳。
- 云同步上传失败写入日志，页面显示可点击日志路径；修复云端状态查询 JSON 解析。

### 修复

- 远程目录双击偶发无法进入子文件夹。
- 异步刷新后远程列表点击失效。
- 行内重命名输入框显示不全。
- 云同步确认框被同步窗口遮挡。
